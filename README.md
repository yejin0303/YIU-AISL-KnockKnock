KnockKnock

A Learning Management Web Application for Efficient Study

KnockKnock is a web application designed to help users create and organize their own study notes, review important concepts, and track learning progress.

⚠️ This repository is a personal portfolio fork of the original team project developed in the AI Service Laboratory.

📖 Overview

KnockKnock is a learning management web application that enables users to create, manage, and review their own study notes efficiently.

The application provides an integrated learning experience by offering note management, quizzes based on user-created notes, memorization, and learning record features within a single platform.
